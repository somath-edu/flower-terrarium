# MathWiki Librarian Schema (CLAUDE.md)

This document defines the core principles and workflows for the **MathWiki** K-12 Digital Math Textbook.

## 1. Core Workflow (Memory Lifecycle)
**Raw (/raw) → Vault (/vault) → Web (/web)**

- `/raw`: For incoming raw source materials.
- `/vault`: For organized knowledge (Obsidian Markdown).
- `/web`: For digital publishing (Astro).

## 2. Design Principles
- **Analog Style**: EB Garamond typography, cream paper textures.
- **Interactive Math**: High-fidelity visualizations using Mafs and JSXGraph.

## 3. Mission
"To refine raw mathematical information into a structured knowledge base and visualize it as a premium-quality analog-style digital textbook."
